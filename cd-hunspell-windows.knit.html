hunspell 사전

hunspell 한국어 사전 GitHub 저장소에서 다운로드 받는다.

Sublime Text 3 1

Sublime Text 3 편집기에서 Hunspell 한국어 맞춤법 검사를 하기 위해서 먼저 사전을 설치해야 된다. 설치방법은 다음과 같다. Dictionaries 한국어 사전이 다소 오래된 hunspell 0.7.1 3년전 사전이 등록되어 있다.

  1. Ctrl-shift-p 를 눌러 Package Control: Add Repository 선택한다.
  2. https://github.com/titoBouzout/Dictionaries URL을 붙여넣는다.
  3. Package Control: Install Package 에서 Dictionaries가 보이기 때문에 설치하면 된다.

설치가 완료되면 View → Dictionary → Dictionaries → Korean을 순차적으로 선택하여 맞춤법 검사를 시작한다.

  1. Sublime Forum, “Can’t install new dictionaries”↩︎
